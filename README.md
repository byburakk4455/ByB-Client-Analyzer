# ByB-Client-Analyzer

**This tool scans your JAR files using AI and checks for any webhooks or data transfers, essentially performing a RAT scan. It also analyzes which parts of the client were created manually and which were created manually, providing percentages of whether the client was created manually or with AI. Furthermore, it analyzes the client and provides a description.**
