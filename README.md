# ByB-Client-Analyzer

🇺🇸 **This tool scans your JAR files using AI and checks for any webhooks or data transfers, essentially performing a RAT scan. It also analyzes which parts of the client were created manually and which were created manually, providing percentages of whether the client was created manually or with AI. Furthermore, it analyzes the client and provides a description.**

🇹🇷 **Bu araç, yapay zeka kullanarak JAR dosyalarınızı tarar ve herhangi bir web kancası veya veri aktarımı olup olmadığını kontrol eder; esasen bir RAT taraması gerçekleştirir. Ayrıca, istemcinin hangi bölümlerinin manuel olarak, hangilerinin ise yapay zeka ile oluşturulduğunu analiz eder ve istemcinin manuel olarak mı yoksa yapay zeka ile mi oluşturulduğuna dair yüzdelik oranlar sunar. Dahası, istemciyi analiz eder ve bir açıklama sağlar.**
